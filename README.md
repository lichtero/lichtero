Maps infrastructure dependencies and technical liabilities across an environment, and untangles tech stack fragmentation left by M&A.

### 2. Cognitive Debt Remediation
Builds delivery discipline that captures the institutional "why" behind architecture and configuration choices, so teams don't hand off hidden context to the next shift.

### 3. Process Harnesses (model-agnostic AI integration)
Gateway middleware that decouples core business logic from specific AI model providers, protecting delivery speed and avoiding single-vendor lock-in.
